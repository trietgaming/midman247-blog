## Xin chào!

Đây là blog site được xây dựng bằng **Next.js** và **Decap CMS**. Bạn có thể quản lý bài viết trực tiếp qua Git mà không cần database phức tạp.

### Các tính năng chính:
- **Tốc độ cực nhanh:** Sử dụng Static Site Generation.
- **Dễ quản lý:** Viết bài bằng Markdown hoặc giao diện Admin.
- **Open Source:** Mọi người có thể đóng góp nội dung qua Pull Request.

Chúc bạn có những trải nghiệm tuyệt vời!
